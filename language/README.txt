This directory contains translated YaPB message file.

If you wish to use one of these translated files, please copy one of these files to cstrike\addons\yapb folder, and rename it to "lang.txt".

Note: Chinese, Japanese and Korean translations cannot be used in CS 1.5 or earlier, as the old versions of HL engine does not support displaying non-latin characters.

---------------------------------------------------------------------
Currently only Chinese and German translations are available, please help if you can do other translations. Thanks in advance.
